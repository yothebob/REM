Learning the Tools

Hello! Here is the in depth guide for all the tools available to you in the Residental Estimating Model! This software was designed with the general contactor in mind and all the tools you will need for rail quoting are at your disposal. If you are a contractor this tool can save you precious time, and also give you the ability to give instant quotes in the field! We have a web application for contractors to use but also have a windows desktop version available for purchase if you want to have it offline on your computer/laptop.

Drawing Tools

There are many drawing tools at your disposal such as any paint software or blueprint pdf viewer. We will go over the basics of each and how to use them most efficiently.

  • Line Tool

    • The line tool will be the one you use most frequently, The line tool is selected when you first enter the program. The line tool by default is set to snap to the grid in half step increments (That means you can draw in the middle of cells as well).You can turn off grid snapping by clicking the "Grid" button on the tool bar, this will also ask if you still want a grid. You can press the button again to change the settings again. When drawing a line you will see a number under the line, that is the measurement of the line given the scale you selected. You can manipulate your line width by pressing "W" or clicking the line width button. You can also change the color of your lines by clicking the "color selector" scroll down tab.

  • Rectangle Tool

    • The rectangle tool acts very simular to the line tool but it draws a rectangle. There is no scale shown as you draw the rectangle, but you can still draw them with diffent widths and colors.If you draw a rectangle you can not undo one side, it will undo the enire retangle.

  • Pen Tool

    • As of now the pen tool is not reccomended for use because it does not support undoing (ctrl + Z). The pen tool does support color and width.
